# Monorepo

This repository is managed as a monorepo using npm workspaces.

## Packages

- **@plain-kit/router**: Located in `packages/router`. A lightweight frontend router using URLPattern.

## Commands

- Install dependencies: `npm install`
- Run tests: `npm test -w @plain-kit/router`
- Build packages: `npm run build -w @plain-kit/router` (if applicable)
